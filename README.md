# Violet Storm – веб-версия 🌐💜

[![HTML5](https://img.shields.io/badge/HTML5-Canvas-orange?logo=html5)](https://developer.mozilla.org/ru/docs/Web/HTML)
[![GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-онлайн-blue?logo=github)](https://pages.github.com)
[![License](https://img.shields.io/badge/license-MIT-brightgreen)](LICENSE)

**Пиксельный 2D-экшен**, портированный на HTML5 Canvas.  
Девочка попадает в лабиринт, сражается со слаймами, плюющимися пузырями, и финальным боссом.  
Игра запускается прямо в браузере – никаких установок!

---

## 🚀 Как открыть игру

### Способ 1: Локально
1. Скачай репозиторий (Code → Download ZIP).
2. Убедись, что все PNG-файлы лежат рядом с `index.html`.
3. Открой `index.html` в любом современном браузере (Chrome, Edge, Firefox).

### Способ 2: Через GitHub Pages (онлайн)
1. Перейди в настройки репозитория (**Settings**).
2. В левом меню выбери **Pages**.
3. В разделе **Branch** выбери `main` (или `master`) и нажми **Save**.
4. Через минуту игра станет доступна по адресу:  
   `https://твой-username.github.io/имя-репозитория/`

> ⚠️ **Важно**: все изображения (`pole.png`, `vrag1.png`, `pula1.png` и т.д.) должны находиться в корне репозитория вместе с `index.html`.

## 🎮 Управление

| Клавиша          | Действие                         |
| ---------------- | -------------------------------- |
| `W` `A` `S` `D` | Движение героини                 |
| Левая кнопка мыши| Выстрел из пистолета             |
| `Enter`          | Войти в портал (когда он активен)|

## 📖 Особенности

- **5 уровней** с уникальной расстановкой стен.
- **Враги-слаймы**: держат дистанцию, стреляют пузырями, анимированы.
- **Финальный босс** (уровень 5) с тремя режимами атаки.
- **Портал** с анимацией – появляется после зачистки уровня.
- Экран смерти и кнопка «ВОЗРОДИТСЯ».
- Счётчик побед (в веб-версии не сохраняется, но показывает текущий результат).
- Пиксельная графика (легко заменяемая на свои файлы).
